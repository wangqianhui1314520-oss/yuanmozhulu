# 元末逐鹿 — 设计语言锁 (V6.0)

## Design Read
**"案牍之上的天下"** — 元帅案头散落着不同材质的文书器物：羽檄交驰的军报竹简、蜡封密函、待批奏章、线装典籍、朱批圣旨。每一类面板都是案上的一件"器物"，而非同一张网页的不同Tab。玩家在"批阅"而非"浏览"。

## 美学方向
**方向：器物差异化 (Artifact Taxonomy)** — 对抗 AI-slop 的"通用卡片风格"：
- 不是所有面板用同一套竹简卡片，而是**六种时代器物**各有其质感
- 军报粗粝紧迫，圣旨雍容威严，密函阴暗隐秘，奏章工整秩序，典籍古雅沉静，人物温润含蓄
- 切换面板 = 在案上拿起不同的文书，而非切换网页标签
- 每件器物有其专属的开合动画（军报：横向疾驰 / 圣旨：竖向展卷 / 密函：蜡封裂开 / 典籍：翻页 / 奏章：铺展朱批 / 人物：屏风推移）

## 六类器物系统

### 1. 圣旨/诏书（`.artifact-edict`）
- **器物原型**：明黄绢帛卷轴 + 玉玺封印
- **底色温偏**：暖金 +3°（`#1e1812`），模拟绢帛微黄
- **边框**：双层鎏金如意纹边框，顶部云纹装饰
- **开合动画**：`edictUnfurl` — 从天垂落竖轴 → 左右展开 → 玉玺落下
- **组件映射**：EndingPanel, TurnSummaryScroll

### 2. 军报/羽檄（`.artifact-dispatch`）
- **器物原型**：竹简 + 火漆封缄 + 鸡毛羽檄
- **底色温偏**：粗粝 + 竹纹水平线（`#1c1712`）
- **边框**：粗糙木纹边，左上角火漆印
- **开合动画**：`dispatchRush` — 横向疾入 + 微颤（军情如火）
- **组件映射**：WarPanel, MarchPanel, BatchRecruitPanel

### 3. 密函/蜡丸（`.artifact-secret`）
- **器物原型**：蜡封密函 + 鱼符 + 暗格
- **底色温偏**：冷暗 -2°（`#15141a`），隐秘幽暗
- **边框**：细窄暗边 + 蜡封质感
- **开合动画**：`sealCrackReveal` — 蜡封印裂开 → 函件从裂缝中展露
- **组件映射**：SecurityPanel, DiplomacyDeepPanel, FactionNetworkPanel

### 4. 典籍/版刻（`.artifact-codex`）
- **器物原型**：线装书 + 木版印刷 + 眉批
- **底色温偏**：象牙白纸感（`#1a1812`），书页微黄
- **边框**：书口线装针脚 + 双色套印边
- **开合动画**：`pageTurn` — 3D翻页，书页从左掀开
- **组件映射**：TechTreePanel, FactionGalleryPage, HistoryAnchorPanel

### 5. 奏章/案牍（`.artifact-memorial`）
- **器物原型**：官员奏章 + 朱批 + 算盘
- **底色温偏**：官纸微青（`#161a18`），仿古纸白
- **边框**：工整官牍框线 + 抬头格式
- **开合动画**：`memorialUnfold` — 奏章铺平 → 朱笔圈画生效
- **组件映射**：PolicyPanel, RecruitPanel, BatchBuildPanel

### 6. 屏风/人物（`.artifact-personnel`）
- **器物原型**：木质屏风 + 茶盏 + 香炉
- **底色温偏**：暖木 +5°（`#1e1814`），木纹温润
- **边框**：木框榫卯 + 铜角装饰
- **开合动画**：`screenSlide` — 屏风面板从右侧推移入位
- **组件映射**：AdvisorPanel, GeneralPanel, GeneralChatPanel, CharacterPortrait, TalentMarketPanel

### 通用/系统类（默认竹简风格，无特殊class）
SettingsPanel, AIControlPanel, ReplayPanel, TutorialOverlay, LoadingOverlay, GameEventOverlay
沿用V5.0竹简卡片默认样式。

## 组件规约（V6.0 扩展）
- 所有交互按钮必须使用 `.btn` / `.btn-gold` / `.btn-danger` 三档体系
- 所有弹窗必须使用 `.modal-panel` 基础样式 + 器物专属动画
- 所有面板类组件必须指定 `artifact-*` 器物类别
- 所有 Toast 统一由 `<AppToast>` 组件渲染
- 器物动画遵循"开合呼应"原则：入场动画对应离场动画的逆过程

## 反AI-Slop禁令（V5.0 继承）
- ❌ 禁止渐变色按钮（紫色渐变、蓝紫渐变等）
- ❌ 禁止发光边框 + 玻璃态叠加（glassmorphism）
- ❌ 禁止无衬线字体 + 大圆角 + 投影卡片（Material Design 克隆）
- ❌ 禁止 emoji 作为 UI 图标
- ❌ 禁止纯黑 `#000` 和纯白 `#fff`
- **V6.0 新增** ❌ 禁止所有面板使用同一套视觉模板（必须器物差异化）
