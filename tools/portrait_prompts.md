# NPC 肖像生成提示词参考 — Phase 1

使用 `image-portrait` Skill 生成肖像时的风格统一 Prompt 模板。
所有肖像要求：512×768 WebP，半身工笔古风，约 150KB。

## 通用风格参数
```
model: nano2 (质量优先)
style: "A" (身份一致角色风格)
format: webp, 512×768
background: 素色/云纹渐变
```

## 风格关键词（全部肖像通用）
```
元末明初历史人物，工笔重彩风格，半身朝服像/戎装像，
古画质感，暖色调，宣纸肌理，不出现现代元素，
中国古代人物画，明代肖像画风，绢本质感
```

## 禁止元素
- 现代服装、眼镜、手表
- 动漫风格、卡通风格
- 3D渲染感、照片感
- 英文文字、阿拉伯数字
