### 基本信息

| 项目 | 内容 |
|------|------|
| 难度 | **中等** ★★★☆☆ |
| 初始领土 | 6地块（太原为中心的山西/陕甘） |
| 银两 | 8,000 |
| 粮草 | 5,000 |
| 兵力 | 4,000 |
| 兵器 | 120 |
| 战马 | 150（全势力第二多） |
| 声望 | 45 |
| 谋士团 | 王保保（1人） |
| Buff | 骑兵战力+40%、将领忠诚度+20% |
| Debuff | 南北同时用兵资源消耗+30%、元廷内部掣肘 |

### 战略定位

王保保是**全势力最强骑兵统帅之一**。骑兵+40%配合150匹初始战马，野战无敌。但两线作战debuff是最大隐患。核心玩法是**铁骑奔袭，速战速决**。

### 核心敌人

- 🟠 徐寿辉（-20，红巾讨伐）
- 🟠 漠北（-15，草原控制权）

### 游玩路线

#### 第一阶段：前10回合 —— 骑兵备战

**目标：集中资源，打造最强骑兵军团**

| 回合 | 操作建议 |
|------|----------|
| 第1回合 | `recruit` 骑兵2,000 + `buy_horses` 100匹 |
| 第2回合 | `train_troops` 太原 + `supply` 太原 |
| 第3回合 | `fortify` 太原 + `scout` 徐寿辉 |
| 第4回合 | `march` 大同方向（统一山西） |
| 第5回合 | `attack` 大同周边 + `garrison` 太原 |
| 第6回合 | `train` 骑兵（强化训练） |
| 第7回合 | `develop` 太原 + `tax` |
| 第8回合 | `scout` 漠北 + `fortify` 大同 |
| 第9回合 | `diplomacy` 元廷（表忠） |
| 第10回合 | 评估南北战线 |

**关键指标**：
- 骑兵占比60%以上
- 战马储备200+
- 太原城防Lv2+

#### 第二阶段：第11~30回合 —— 南征北战

**目标：先南后北，逐个击破**

1. **南下讨徐寿辉**（第11~20回合）：
   - `march` 南阳方向 + `attack` 南阳
   - 徐寿辉初始兵力3,500，不如王保保的骑兵
   - 快速决战，避免持久战（资源消耗+30%）

2. **北上定漠北**（第21~30回合）：
   - `march` 和林方向 + `attack`
   - 漠北也是骑兵势力，骑兵对决硬碰硬
   - 利用骑兵+40% vs 漠北骑兵+45%，略占下风，靠将领忠诚+20%弥补

#### 第三阶段：第31~50回合 —— 扶保元廷

1. 稳定山西-陕甘-河南根据地
2. 帮助元廷对抗朱元璋/陈友谅
3. `diplomacy` 元廷保持关系，避免朝堂猜忌debuff恶化
4. 最终或自立或继续效忠

### 关键决策点

1. **先南还是先北？** → 先南（徐寿辉），南方势力弱且与元廷敌对
2. **要不要脱离元廷？** → 中期之前不要，朝堂猜忌debuff会因脱离而恶化
3. **两线作战怎么避免？** → 打完一边再打另一边，绝不同时开战
4. **战马消耗？** → `buy_horses` 保持储备，骑兵没有马就是步兵
5. **元廷掣肘怎么应对？** → `diplomacy` 定期表忠 + `tribute` 上贡

| 第4回合 | `fortify` 江州 + `buy_horses` 60匹 |
| 第5回合 | `attack` 安庆（如已到） + `farm` 长沙 |
| 第6回合 | `train` 水师 + `garrison` 武昌 |
| 第7回合 | `relief` 武昌（民心！民心！民心！） |
| 第8回合 | `spy` 朱元璋 + `develop` 江州 |
| 第9回合 | `recruit` 2,000人 + `supply` 安庆 |
| 第10回合 | 准备鄱阳湖决战 |

**关键指标**：
- 兵力突破10,000（利用高初始兵力+征兵）
- 水师训练到位
- 民心不低于30

#### 第二阶段：第11~25回合 —— 鄱阳湖决战

**目标：一战定乾坤，消灭朱元璋**

   - 利用水战+50%在鄱阳湖水域决战
   - 朱元璋水战不如陈友谅

2. **攻占应天**（第16~20回合）：
   - `attack` 太平府 → `attack` 应天府
   - 朱元璋核心领土

3. **消灭残部**（第21~25回合）：
   - 扫荡朱元璋残余地块

#### 第三阶段：第26~50回合 —— 横扫江南

1. 东灭张士诚（水师沿长江而下）
2. 收服方国珍（海陆夹击）
3. 北上徐寿辉（历史仇敌，速灭）
4. 西取明玉珍（水路入川）
5. 北伐元廷

### 关键决策点

1. **什么时候打朱元璋？** → 越早越好！不要给朱元璋发展时间
2. **民心暴跌怎么办？** → 每3回合必须 `relief`！这是最大弱点
3. **外交孤立？** → 放弃外交，军事解决一切。唯一可拉拢的是张士诚
4. **水战优势怎么最大化？** → 所有战斗尽量选择沿江/沿海地块
5. **弑主debuff？** → 无法消除，接受它，用军事优势碾压
6. **鄱阳湖锚点**：触发后有3个分支，选择不同战术（火攻/正面/奇袭）
